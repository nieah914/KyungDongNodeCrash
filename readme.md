    ```
    git pull origin master
    ```

## 2020-07-19 업데이트
### 프로젝트 백엔드 기본 특 작성 완료
* 프로젝트 사용법은 금주 스터디 시간에 간단히 설명 예정
* 프로젝트 내려받기
    ```
    첫 프로젝트 내려받기
    git clone https://github.com/nieah914/KyungDongNodeCrash.git
    ```
    상단의 명령어를 이용해서 내려받음


* 웹서버 시작 방법
   ```
    npm run dev 
   ```

* 수정한 프로그램 반영시
    * 수정 사항을 확인
    ```
    git status
    ```
   
    * 수정할 사항들을 타켓팅
    ```
    git -u
    git add *
    ```

    * 수정할 사항들을 commit
    ```
    git commit -m "20200719 정해인 - 프로젝트 틀 반영"
    ```
